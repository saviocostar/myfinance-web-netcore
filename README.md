## Tecnologias
- Microsoft SQL

<br>

## Comandos Úteis
A aplicação está dentro da pasta src, para rodar a aplicação utilize o comando :

- `dotnet build` - build da aplicação
- `dotnet run` - para rodar
- `dotnet watch run` - rodar com live update
